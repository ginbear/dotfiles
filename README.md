## Set up for MacOS

1. 適当な場所で git clone して持ってくる ( リポジトリは後ほど ghq 管理する
2. `sh bin/setup_mac.sh`
3. `sh bin/setup_homebrew.sh`
4. `sh bin/setup_homebrew_cask.sh`
5. `ghq get https://github.com/ginbear/dotfiles.git`
6. `ghq look dotfiles`
7. `sh bin/init.sh`
